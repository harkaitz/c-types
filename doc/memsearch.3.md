# NAME

memsearch()

# SYNOPSIS

    #include <str/memsearch.h>
    
    size_t memsearch(const char _m[], size_t _msz, const char _s[], bool _shift);

# DESCRIPTION

Search a string in a memory block.

# RETURN VALUE

Returns the position in the buffer if found, otherwise returns -1.
